INFORMATION

Global Skin - is highly realistic skin shader which can be applied onto several different Character Creator Models within Blender. This allows you to make a Realistic Character very quickly. But there is an issue.

Currently Global Skin requires you to Copy and Paste each individual value for each effect from one material into every other material on your character. This is a very time consuming process. So I have developed an addon to speed up your workflow.

Universal Controls for Global Skin - is an addon I designed which allows you to customise all four Global Skin body part materials at once, by adding control groups that are all linked together and connected to Global Skin's Shader menu.

When you open one of the control groups, you will find every recreated Global Skin value, colour coded and marked so you can quickly make changes.

Other features include a quick Apply Global Skin to all Materials button and a Save Template button. You can also make the Control Groups and Global Skin Materials unique, if you want to work on multiple characters at once.
